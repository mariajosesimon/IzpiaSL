config module¶

config.PWD = 'D:\\3DAM\\Practicas con Fernando\\Electricidad_v1\\docs'¶

Arguments:

USERNAME {string} – nombre de usuario password {string} – Contraseña de la base de datos HOST {string} – dirección local PORT {integer} – Número de puerto DATABASE {string} – Nombre de la base de datos SQLALCHEMY_DATABASE_URI {string} – Cadena de conexion a la base de datos
